About This Project

📊Project Overview

This project delivers an end-to-end analytics portal for XYZ Retail Chain, integrating Tableau dashboards with a custom-built web interface to provide actionable insights into sales and operational quality.

🗂Data Sources

  • xyz_Sales.xlsx – Sales data
  • xyz_Quality.xlsx – Quality metrics

⚙Assumptions

  • Data is assumed to be clean and accurate
  • Monthly aggregation used for KPIs
  • Consistent outlet identifiers
  • Dashboards embedded using Tableau Public
